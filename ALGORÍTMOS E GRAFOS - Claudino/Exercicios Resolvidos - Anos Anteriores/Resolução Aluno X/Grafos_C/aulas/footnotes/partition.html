Uma partição de um conjunto X é uma coleção

X1,  X2,  … ,  Xk

de subconjuntos não-vazios de X tal que cada elemento de X pertence a um e apenas um Xi. 

Cada Xj é uma parte ou bloco da partição.  (É muito errado dizer que "Xj é uma partição" e que "Xj é uma das partições".)

Se k = 2, diz-se que temos uma bipartição.
