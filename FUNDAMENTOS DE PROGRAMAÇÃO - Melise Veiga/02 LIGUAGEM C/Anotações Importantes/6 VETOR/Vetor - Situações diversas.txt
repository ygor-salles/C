VETOR COM USO DE UM FOR DENTRO DE OUTRO: Deve-se trocar a variável contador, para que seja referenciado
a posição do vetor que você faz referencia.
EX: Faça um programa que preencha dois vetores, X e Y, com dez números inteiros cada. Quantos elementos são iguais
dentre os dois vetores, independente da posição?
1º vetor X: 2 5 3 6 4 1 9 10 0 -2
2º vetor Y: 1 -4 12 8 2 5 3 0 3 7 

int x[10], y[10], i, j, c1=0;
for (i=0; i<10; i++){
	for (j=0; j<10; j++){
		if (x[i] == y[j]){
			c1++;
		}
	}
}
printf ("\nA quantidade de valores iguais eh: %d",c1);
