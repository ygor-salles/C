ACENTUAÇÃO EM C: Para uso da acentuação é necessário incluir a biblioteca
local.h e inserir o comando setlocale(LC_ALL, "Portuguese") dentro da
função principal, conforme código abaixo:

#include <stdio.h>
#include <stdlib.h>
#include <locale.h>
int main(void)
{
  setlocale(LC_ALL, "Portuguese");
  printf("Utilizando caracteres e acentuação da língua portuguesa!\n\n");

  system("pause");
  return 0;
}
